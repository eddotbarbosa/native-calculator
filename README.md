# Native Calculator
A calculator with common operations.

![Project Preview](./gitassets/project-preview.png)

## Features
* common calculator operations
* dark mode

## Technologies
* expo
* react-native
* typescript

## Getting Started
### installation:
```
git init
https://github.com/eddotbarbosa/native-calculator.git
npm install
```
### running:
start
```
npm start
```
android
```
npm run android
```
ios
```
npm run ios
```
web
```
npm run web
```
